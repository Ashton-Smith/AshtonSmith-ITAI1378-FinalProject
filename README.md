# ITAI 1378 Final Project

## Overview

This repository contains my final project work for ITAI 1378 Computer Vision at Houston City College. The project includes computer vision exercises, convolutional neural network experiments, image processing activities, AI workflow development, and notebook based machine learning projects completed throughout the semester using Python, Jupyter Notebook, and Google Colab.

The purpose of this repository is to organize and document the projects, experiments, research, and outputs developed during the course while demonstrating practical understanding of computer vision and artificial intelligence concepts.

---

## Repository Structure

### notebooks
Contains the Jupyter notebooks used throughout the semester including image processing labs, CNN projects, workshop exercises, classification activities, and AI development work.

### docs
Contains project documentation, reports, AI usage logs, presentation materials, and supporting written content related to the final project.

### results
Contains screenshots, outputs, processed images, visualizations, and testing results generated from notebook experiments and computer vision workflows.
