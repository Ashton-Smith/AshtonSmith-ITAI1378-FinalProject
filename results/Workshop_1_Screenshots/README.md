This folder contains screenshots and output examples from the Workshop 1 neural network image classification project. The project focused on training a convolutional neural network to distinguish between chihuahuas and muffins using Python, PyTorch, and Google Colab. The screenshots document different stages of the workflow including dataset preparation, image preprocessing, dataloader configuration, model training, validation accuracy, prediction outputs, and neural network performance analysis. These results demonstrate hands on experimentation with computer vision concepts, deep learning pipelines, and image classification techniques learned throughout ITAI 1378.
